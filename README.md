# FluxInsight

FluxInsight is a testimonial collection and management platform designed for B2B businesses to gather, organize, and showcase customer testimonials efficiently.

## 🚀 Features

* Responsive landing page
* Testimonial submission system
* Client dashboard
* Provider dashboard
* Authentication system
* Testimonial approval workflow
* Search and filter testimonials
* Mobile-friendly UI

## 🛠 Tech Stack

* React
* Vite
* Tailwind CSS
* Material Tailwind
* Firebase (Authentication + Firestore)
* React Router

## 📂 Project Structure

src/
├── assets/
├── components/
├── pages/
├── routes/
├── services/
├── hooks/

## ⚙️ Installation

Clone the repository:

```bash
git clone <your-repo-url>
```

Move into project folder:

```bash
cd FluxInsyte
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
