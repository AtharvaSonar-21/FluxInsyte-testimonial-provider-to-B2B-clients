# FluxInsyte — Testimonial Provider for B2B Clients

FluxInsyte is a modern SaaS platform built to help freelancers, students, and B2B service providers collect, organize, and manage client testimonials professionally.

It simplifies testimonial collection through a structured dashboard, branded request flow, and centralized storage.

---

## 🚀 Project Vision

Client testimonials are often scattered across chats, emails, or lost completely.

FluxInsyte solves this by providing a clean system where users can:

- request testimonials through a dedicated link  
- receive responses instantly  
- manage testimonials inside a dashboard  
- present social proof professionally

## ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/AtharvaSonar-21/FluxInsyte---testimonial-provider-to-B2B-clients.git
````

Move into folder:

```bash
cd FluxInsyte---testimonial-provider-to-B2B-clients
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
